# 🌟 Projeto Agrinho 2026 - Proteção de Nascentes

Este projeto foi desenvolvido para o **Programa Agrinho 2026**, com o objetivo de conscientizar a comunidade sobre a importância da preservação das nascentes de água e oferecer um guia prático de como protegê-las.

## 🚀 Tecnologias Utilizadas
* HTML5
* CSS3

## 💧 Funcionalidades do Site
* **Conscientização:** Explicação simples sobre o que são as nascentes.
* **Passo a Passo:** Guia prático de como proteger uma nascente (cercamento, plantio de nativas, etc.).
* **Importância:** Dados sobre o impacto da água na agricultura e na vida.
